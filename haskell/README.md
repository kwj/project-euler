# Solutions in Haskell

## How to run

```console
sh solve.sh <problem_number ...>
```

The following is an examples of execution.

```console
sh solve.sh 1
```

## Notes

I have confirmed that it works in the following environments.
  - GHC 9.12.2 / 9.14.1
  - Cabal 3.16.1.0

Please see `euler.cabal` for dependent packages.
