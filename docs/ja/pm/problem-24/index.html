感想

辞書順で返してくれる順列がライブラリ等にあれば 100 万回ぶん回すのも手段の一つですが、 せっかくの辞書順なので素直に考えてみます。

順列の総数は $10! = 3628800$ なので、辞書順ならば最初が 0 から始まるのが $9! = 362880$ 個、 次に 1 から始まるのが 362880 個という形になる。つまり $1000000 / 362880 = 2.75…$ なので 問題の順列は 3 番目の 2 から始まることが判る。次は 2 を除いた $0, 1, 3, 4, 5, 6, 7, 8, 9$ で始まるのが $8! = 40320$ ずつあるので、$1000000 \equiv 27420 \enspace (mod \; 362880)$ から 274240 番目は $274240 / 40320 = 6.80…$ となり 7 番目の 7 から始まることが判る。以下、繰り返し。

木構造を上からたどっていると考えれば理解しやすいかな。
