Problem 27

問題へのリンク

  • Quadratic primes

感想

ある程度、範囲を絞り込める。

$n=0$ の場合から b は素数でなければならない。同様に $n=1$ の場合から $1+a+b$ も素数になる。 $x=1+a+b$ とおくと $a=x-b-1$ となるので $|x-b-1| < 1000$。$2 \le b < 1000$ より $0 < x < 2000$。

あとは 1000 以下の素数 $b$ と 2000 未満の素数 $x$ から求められる条件を満たす $a$ の組み合わせにおいて、 式 $n^{2}+an+b$ で素数がどれだけ続くかを確認すれば良い。
