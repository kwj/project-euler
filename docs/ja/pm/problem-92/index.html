Problem 92

問題へのリンク

  • Square digit chains

感想

1000 万未満の全て数字が 1 回の変換で 3 桁になるし、 3 桁の数字は 1 回の変換で全て 243 以下になる。

  9^2 = 81
  -->
        99 -> 9^2 * 2 = 162
       999 -> 9^2 * 3 = 243
      9999 -> 9^2 * 4 = 324
     99999 -> 9^2 * 5 = 405
    999999 -> 9^2 * 6 = 486
   9999999 -> 9^2 * 7 = 567

仮に 567 まであらかじめ調べてテーブルにしておけば、変換 1 回で最終結果が判明する。
